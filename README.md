# A&A Design - Telegram Web App

Веб-приложение для Telegram Bot магазина кастомных журналов A&A Design.

## 🎨 О проекте

A&A Design специализируется на создании персонализированных журналов - уникальных подарков, которые сохраняют воспоминания в красивой печатной форме.

## 🚀 Технологии

- HTML5
- CSS3
- JavaScript (Vanilla)
- Telegram Web App API
- Python (Telegram Bot)

## 📱 Особенности

- Адаптивный дизайн для мобильных устройств
- Интеграция с Telegram Bot API
- Минималистичный и уютный интерфейс
- Цветовая схема: #FFBDC8, #F00B0D, #FFE3E7
- Прямое перенаправление к менеджеру при заказе

## 🔧 Установка и запуск

### Web App
1. Разместите файлы на веб-сервере
2. Обновите URL в настройках Telegram Bot

### Telegram Bot
1. Установите зависимости:
```bash
pip install python-telegram-bot
```

2. Настройте переменные окружения:
```bash
export BOT_TOKEN="your_bot_token"
export WEB_APP_URL="https://your-domain.com"
export ADMIN_ID="your_admin_id"
```

3. Запустите бота:
```bash
python bot.py
```
