# A&A Design - Telegram Web App с 3D книгами

Веб-приложение для Telegram Bot магазина кастомных журналов A&A Design с интерактивными 3D кейсами.

## 🎨 О проекте

A&A Design специализируется на создании персонализированных журналов - уникальных подарков, которые сохраняют воспоминания в красивой печатной форме.

**🆕 Новинка:** Интерактивные 3D книги с кейсами! Нажмите на книгу, чтобы перелистнуть страницы и изучить примеры работ.

## 🔧 Установка и запуск

### Требования к изображениям 3D книг:
- **Формат**: PNG (обязательно)
- **Размер**: 400px × 560px (рекомендуемый)
- **Качество**: Высокое разрешение
- **Количество**: 16 PNG файлов для 4 книг

### Web App
1. Разместите файлы на веб-сервере
2. Загрузите PNG изображения в папку `assets/images/cases/`
3. Обновите URL в настройках Telegram Bot

### Telegram Bot
1. Установите зависимости:
```bash
pip install python-telegram-bot
```

2. Настройте переменные окружения:
```bash
export BOT_TOKEN="your_bot_token"
export WEB_APP_URL="https://your-domain.com"
export ADMIN_ID="your_admin_id"
```

3. Запустите бота:
```bash
python bot.py
```

## 🛠 Техническая архитектура

### JavaScript модули:
- `TelegramApp` - интеграция с Telegram Web App API
- `Navigation` - система навигации с историей
- `Books3D` - **модуль для 3D книг**
- `Shop` - функциональность магазина
- `ReviewsSlider` - слайдер отзывов (временно отключен)
- `FAQ` - аккордеон вопросов
- `Contacts` - обработка контактных ссылок
- `Analytics` - отслеживание взаимодействий
- `Utils` - вспомогательные функции

### CSS особенности:
- **3D трансформации** с `transform-style: preserve-3d`
- **Перспектива** с `perspective: 1000px`
- **Плавные анимации** с `cubic-bezier` функциями
- **Адаптивные медиа-запросы** для всех устройств
- **Кастомные CSS переменные** для тематических цветов

## 🎨 Кастомизация 3D книг

### Добавление новой книги:
1. Создайте HTML структуру в `index.html`
2. Добавьте PNG изображения (4 файла на книгу)
3. Настройте CSS стили в `styles.css`
4. Обновите JavaScript в `main.js`

### Настройка анимаций:
- Измените `transition` для скорости анимации
- Настройте `transform` для углов поворота
- Обновите `box-shadow` для теней
