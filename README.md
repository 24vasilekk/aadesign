# A&A Design - Telegram Web App с 3D книгами

Веб-приложение для Telegram Bot магазина кастомных журналов A&A Design с интерактивными 3D кейсами.

## 🎨 О проекте

A&A Design специализируется на создании персонализированных журналов - уникальных подарков, которые сохраняют воспоминания в красивой печатной форме.

**🆕 Новинка:** Интерактивная 3D книга с 8 разворотами! Нажмите на книгу, чтобы перелистнуть все страницы и изучить примеры работ.

## 🚀 Технологии

- HTML5
- CSS3 с 3D трансформациями
- JavaScript (Vanilla) с модульной архитектурой
- Telegram Web App API
- Python (Telegram Bot) с расширенной аналитикой

### Web App
1. Разместите файлы на веб-сервере
2. Загрузите PNG изображения в папку `assets/images/cases/`
3. Обновите URL в настройках Telegram Bot

### Telegram Bot
1. Установите зависимости:
```bash
pip install python-telegram-bot
```

2. Настройте переменные окружения:
```bash
export BOT_TOKEN="your_bot_token"
export WEB_APP_URL="https://your-domain.com"
export ADMIN_ID="your_admin_id"
```

3. Запустите бота:
```bash
python bot.py
```
